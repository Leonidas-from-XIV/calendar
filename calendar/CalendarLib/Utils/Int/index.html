type t = int
val equal : t -> t -> bool

Equality over t.

val compare : t -> t -> int

Comparison over t. compare x y returns 0 iff equal x y = 0. If x and y are not equal, it returns a negative integer iff x is lesser than y and a positive integer otherwise.

val hash : t -> int

A hash function over t.
